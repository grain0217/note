#### postion
postion:absolute，相对于最近的"positioned"祖先元素定位(即static除外)；

#### 垂直居中
1.宽高固定
```css
.parent{
  position:relative;
}
.child{
  width:300px;
  height:300px;
  position:absolute;
  margin-left;-150px;
  margin-top:-150px;
}
```
2.宽高未知
```css
.parent{
  position:relative;
}
.child{
  width:300px;
  height:300px;
  position:absolute;
  transform:translate(-50%, -50%);
}
```
3.flex
```css
.parent{
  display:flex;
  justify-content:center;
  align-item:center;
}
```
